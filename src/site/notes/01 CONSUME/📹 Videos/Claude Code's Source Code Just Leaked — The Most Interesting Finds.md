## Summary
Anthropic accidentally shipped source maps in their NPM package, leaking Claude Code's entire TypeScript source. I read all 380,000 lines — including "undercover mode," which tells Claude to pretend to be human.

The most controversial find: auto-activated for all public repos, undercover mode strips AI attribution from commits. The prompt literally says "Don't blow your cover." How many open source PRs have already gone through like this?

Also inside: Kairos (autonomous Claude Code that works while you sleep), a Tamagotchi pet system, AI that dreams about your code, multi-agent swarms, and anti-distillation defenses that inject fake tools to poison competitor training data.

NOT the model weights — just the CLI/agent harness code.
