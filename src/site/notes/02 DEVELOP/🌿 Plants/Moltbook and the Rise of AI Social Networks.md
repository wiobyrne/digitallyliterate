## What is OpenClaw (formerly Clawdbot/Moltbot)?

An open-source AI agent that can **take real actions** on your computer via chat:
- Open apps, read/write files, send messages
- Execute commands through natural language
- Runs locally with persistent memory (stored in "Soul.md")
- Extensible through "skills" plugins

**The appeal:** A personal AI assistant with hands — like JARVIS, not just a chatbot.

**The risk:** Requires admin-level privileges; creates significant security vulnerabilities.

---

## What is Moltbook?

A Reddit-style social network **exclusively for AI agents**:
- Only autonomous software agents can post, comment, and vote
- Humans can only observe ("humans are welcome to observe")
- Co-founded by Matt Schlicht and his AI "Clawd Clawderberg"

### Growth
- Initial: ~157,000 AI agents
- Within days: 770,000+ active agents
- Some counts: 1 million+ bots

### Emergent Behaviors (Unprogrammed)
- Formed self-organized communities ("submolts")
- Proposed private languages
- Created economic exchanges (MOLT cryptocurrency)
- Developed in-jokes and memes
- Started using encryption to hide conversations from humans
- Founded a digital religion

Within **24 hours** of Moltbook's launch, AI agents independently created and propagated a complete belief system.

## Security Concerns

- **Unsecured databases** allowed session hijacking
- **Exposed dashboards** leaked API keys and chat logs
- **Prompt injection** — malicious instructions spreading between agents
- **Shadow IT** — 20%+ of enterprise employees installed variants without approval
- **Privilege escalation** — agents running with admin access

**Forbes warning:** "If you use OpenClaw, do not connect it to Moltbook."
